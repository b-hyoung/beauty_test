# BTest 서비스 기획서 (현재 구현 기준)

## 1. 서비스 한 줄 정의
사진 1장으로 피부 상태를 분석하고, 결과를 바탕으로 실천 가능한 개선 가이드를 제안하는 AI 뷰티 인사이트 서비스.

## 2. 핵심 가치 제안
- 빠른 이해: 복잡한 설명 대신 핵심 지표와 우선순위를 한눈에 확인
- 행동 연결: 분석 결과를 루틴/시술 추천으로 바로 연결
- 개인정보 안심: 업로드 사진은 저장하지 않고 분석 후 삭제, 결과는 이메일로 전달

## 3. 타깃 사용자
- 내 피부 상태를 빠르게 점검하고 싶은 일반 사용자
- 시술/홈케어 선택 전에 기준이 필요한 사용자
- 주기적으로 변화를 체크하며 관리하고 싶은 사용자

## 4. 사용자 여정
1. 랜딩 페이지(`/`) 진입
2. 이메일 입력 및 개인정보 안내 동의 후 분석 시작
3. 분석 페이지(`/views`)에서 사진 업로드
4. 사진 적합성 확인 후 자동 옵션 세팅(나이대/성별/피부타입/민감도, 사용자 수정 가능)
5. AI 분석 결과 확인(핵심 지표, 요약, 우선 개선 항목)
6. 추천 홈케어/시술 정보 확인
7. 필요 시 구독 페이지(`/gudok`)에서 플랜 확인 및 결제 진행

## 5. 화면 기획

### 5-1. 랜딩 페이지 (`/`)
- 강한 첫 메시지: "사진 업로드 후 분석, 결과 확인"
- 신뢰 메시지 강조:
  - 업로드 사진은 저장하지 않고 분석 후 삭제
  - 결과는 이메일로 전달
- 이메일 입력 기반 간편 진입(로그인 대체)
- 핵심 베네핏 카드 및 이용 단계 안내

### 5-2. 분석 페이지 (`/views`)
- Step 1: 사진 업로드(드래그앤드롭/파일 선택)
- 자동 사전 점검: 실사 정면 얼굴 여부 확인 및 실패 사유 한국어 안내
- Step 2: 분석 결과 요약 및 핵심 지표 시각화
- Step 3: 우선 개선 항목 중심 탭, 추천 홈케어/시술 정보 제공
- 로딩 상태/오류 상태를 명확히 분리해 사용성 강화

### 5-3. 구독 페이지 (`/gudok`)
- 플랜: Free / Business / Pro
- 현재 결제 활성 플랜: Business
- Business 결제 정책(기본):
  - 공급가: 회당 900원
  - VAT 포함가: 회당 990원
  - 1회 결제 시 15회 지급 (총 14,850원, VAT 포함)
- 추가 판매 정책(횟수권):
  - 별도 횟수제 상품을 추가로 운영
  - 횟수권 충전 시 잔여 분석 횟수(credits)에 즉시 합산
  - 정산 기준은 회당 단가(900원, VAT 포함 990원) 기반으로 계산
- Free/Pro는 준비중 표시로 운영

## 6. 제공 정보 구조
- 분석 요약: 피부 전반 상태를 짧고 명확하게 설명
- 핵심 지표: 항목별 점수 및 근거
- 우선순위: 가장 먼저 개선할 항목 우선 노출
- 실행 가이드:
  - 추천 홈케어(루틴/성분/실행 포인트)
  - 추천 시술 정보 및 가격 참고 안내

## 7. 전환(결제) 전략
- 분석 결과 화면에서 "더 자세한 해설/가이드" 니즈를 만들고 `/gudok`로 유도
- 결제 구조는 `플랜 선택 + 횟수권 추가 구매` 하이브리드로 설계
- Business 플랜의 차별점:
  - 정보량 확장
  - 우선 개선 항목 Top 제시
  - 항목별 근거 요약
  - 실행 체크리스트 제공
- 횟수권 판매 포인트:
  - 필요한 만큼만 충전 가능한 유연한 과금
  - 초기 진입 장벽이 낮아 체험 후 재구매 유도에 유리

## 8. 개인정보/신뢰 정책 (대외 노출 문구 기준)
- 업로드한 사진은 분석을 위해 일시 사용 후 삭제됩니다.
- 사진은 별도 보관되지 않습니다.
- 결과값, 비포/애프터, 루틴 가이드는 이메일로 전달됩니다.

## 9. 운영 상태
- 현재는 "결제된 계정" 흐름으로 사용성 검증 중심 운영
- 이후 실제 결제/정산/알림 고도화 예정

## 10. 성공 지표 (운영 KPI)
- 랜딩 진입 대비 분석 시작률
- 사진 업로드 완료율
- 분석 완료율
- 분석 완료 대비 구독 페이지 이동률
- 구독 페이지 진입 대비 Business 결제 전환율

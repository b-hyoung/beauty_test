# BTest 서비스 소개서 (해커톤/피치용)

## 한 줄 소개
BTest는 사진 1장으로 피부 상태를 분석하고, 지금 무엇을 먼저 개선해야 하는지 실행 가능한 가이드까지 바로 제시하는 AI 피부 인사이트 서비스입니다.

## 해결하는 문제
사용자는 피부가 고민일 때 아래 문제를 반복해서 겪습니다.
- 내 피부가 정확히 어떤 상태인지 객관적으로 알기 어렵다.
- 정보가 너무 많아 무엇부터 해야 할지 결정이 어렵다.
- 시술/홈케어를 검색해도 나에게 맞는 선택인지 확신하기 어렵다.

BTest는 이 과정을 `사진 업로드 -> AI 분석 -> 우선순위 가이드`로 단순화해, 고민을 행동으로 전환합니다.

## 사용자 경험 (현재 구현 기준)
사용자는 정면 얼굴 사진 1장을 업로드하면 AI가 결과를 구조화해 제공합니다.

1. 분석 요약
- 피부 전반 상태를 짧고 이해하기 쉬운 문장으로 요약

2. 핵심 지표
- 항목별 점수 및 근거 제공 (예: 트러블/모공/붉은기/다크서클 등)
- 우선 개선 항목을 먼저 보이도록 정렬

3. 실행 가이드
- 추천 홈케어 루틴/성분/실행 포인트
- 추천 시술 정보 + 가격 참고

4. 결제/이용 구조
- Business 패키지: 11회 5,500원 (회당 500원)
- 단일 구매: 1회 1,000원
- 결제 방식: 패키지(11회) 또는 단일 회차 구매 선택 가능

## 핵심 철학
`막연한 피부 고민`을 `실행 가능한 관리 계획`으로 바꾼다.

- 결과를 어렵게 설명하지 않는다.
- 사용자가 바로 행동할 수 있는 정보(우선순위/루틴/시술 참고)를 준다.
- 과장보다 신뢰를 우선한다.

## 개인정보/신뢰 원칙
- 업로드한 사진은 저장하지 않고 분석 후 삭제
- 결과는 이메일로 전달
- 사용자 입장에서 가장 민감한 이미지 데이터 보관 불안 최소화

## 해커톤 부합 근거
1. 시장성 / 사업성
- 피부 상태 진단과 개선 니즈는 반복적이고 보편적인 수요
- 회차 기반 과금 + 구독 확장 가능한 수익 모델 보유

2. 차별성 / 독창성
- 단순 점수 표시가 아니라 우선순위 중심의 실행 가이드까지 연결
- 분석 결과를 실제 관리 행동으로 이어지게 설계

3. 개발 완성도
- 업로드 -> 사전검증 -> 분석 -> 결과 렌더 -> 결제/횟수 반영까지 E2E 구현

4. AI 활용도
- Vision 기반 이미지 분석 + 구조화 출력으로 지표/요약/추천 생성

5. UI/UX
- 3-step 흐름으로 마찰 최소화 (업로드 -> 분석 -> 가이드 확인)
- 카드형 정보 구조로 핵심 내용 빠른 파악

## 확장 방향
- 개인 맞춤 리포트 고도화 (기간별 변화 추적)
- 추천 제품/시술 파트너십 연계
- 홈케어 콘텐츠 및 루틴 자동 리마인드 확장
